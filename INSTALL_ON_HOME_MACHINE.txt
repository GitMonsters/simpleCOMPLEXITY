  WORM PYTHON - INSTALLATION ON YOUR HOME MACHINE (192.168.1.218)
═══════════════════════════════════════════════════════════════════════

When you get home, run ONE of these commands:

────────────────────────────────────────────────────────────────────────
OPTION 1: One-Line Installation (Recommended)
────────────────────────────────────────────────────────────────────────

bash <(curl -s https://raw.githubusercontent.com/GitMonsters/COMPLEXITY/claude/create-test-branch-011CUMhAyjZeoPTf9EWYHRE3/scripts/remote_install.sh)

OR if you have the repo already:

cd ~/COMPLEXITY && git checkout claude/create-test-branch-011CUMhAyjZeoPTf9EWYHRE3 && ./scripts/install.sh

────────────────────────────────────────────────────────────────────────
OPTION 2: Step-by-Step Installation
────────────────────────────────────────────────────────────────────────

# 1. SSH into your home machine
ssh worm@192.168.1.218

# 2. Clone the repository (if not already there)
git clone https://github.com/GitMonsters/COMPLEXITY.git
cd COMPLEXITY

# 3. Switch to the Worm Python branch
git checkout claude/create-test-branch-011CUMhAyjZeoPTf9EWYHRE3

# 4. Install
./scripts/install.sh

# 5. Verify
worm --version
worm --info

────────────────────────────────────────────────────────────────────────

After Installation
────────────────────────────────────────────────────────────────────────

Try these commands:

# Run the comprehensive demo
worm demo/comprehensive_demo.py

# Run the interactive menu
./demo/run_demo.sh

# Test network blocking
worm -c "import socket"  # Should fail

# Test that Python works
worm -c "import json; import sys; sys.stdout.write('✓ Works!\n')"

# Run attack simulation
worm demo/attack_simulations/network_exfiltration.py

# Verify source integrity
python3 tools/ioc_monitor.py --verify-worm-source

Troubleshooting
────────────────────────────────────────────────────────────────────────

If you get "worm: command not found":

export PATH="$HOME/.local/bin:$PATH"

Then add to ~/.bashrc:
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
